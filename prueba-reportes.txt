Al enviar un reporte mediante la ruta POST /reportes, la información se almacena en el array reportes. Cuando se consulta la ruta GET /reportes, el servidor devuelve todos los reportes registrados en formato JSON. Esto permite visualizar la información almacenada mientras el servidor permanece encendido.

Ejemplo de respuesta:

[
{
"id": 1,
"tipo": "Infraestructura",
"descripcion": "Daño en alumbrado público"
}
]
